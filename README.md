# BeBitter - Portfolio

Portfolio pessoal de Bernardo Gomes - Desenvolvedor Full-Stack e Designer.

## 🚀 Tecnologias

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Radix UI
- React Router

## 📦 Instalação

```bash
# Instalar dependências
pnpm install

# Rodar em desenvolvimento
pnpm dev

# Build para produção
pnpm build

# Preview da build
pnpm preview
```
